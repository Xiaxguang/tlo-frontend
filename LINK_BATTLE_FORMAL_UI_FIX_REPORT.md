# 卡牌連線討伐戰第一個正式版 UI 修正報告

## 修正目標

本次修正把原本偏測試版的「格子盤面」調整成第一個正式版方向，重點修正玩家回報的三個問題：

1. 卡牌操作區改成堆疊式盤面，呈現接近麻將連連看 / 上海麻將的層級感。
2. 卡牌尺寸縮小，避免手機畫面過度擁擠。
3. 卡牌圖片不再被邊框裁切，並移除卡牌名稱文字，操作區只顯示卡圖。

## 已修改檔案

- `frontend/tlo-frontend-main/js/link-battle.js`
- `frontend/tlo-frontend-main/css/link-battle.css`
- `frontend/tlo-frontend-main/index.html`

## 主要修改內容

### 1. 操作區改為堆疊式渲染

原本：

- 每一層獨立顯示成 grid。
- 顯示「第 X 層」文字。
- 看起來像工程測試表格。

現在：

- 所有 layer 疊在同一個 board 內。
- 每張卡使用 absolute positioning。
- 依照 layer / row / col 計算 left / top / z-index。
- 上層卡牌會視覺上壓在下層卡牌上。
- 被上層覆蓋的卡牌會變暗且不能點擊。

### 2. 卡牌尺寸縮小

前端會依照操作區寬度與 board 欄數自動計算卡牌大小。

目前限制：

- 最小寬度：約 34px
- 最大寬度：約 58px
- 高度依比例計算

這樣在手機畫面上不會像之前一樣卡牌過大。

### 3. 移除卡牌名稱文字

原本卡牌下方會顯示 card_name，現在已移除。

保留：

- `title`
- `aria-label`
- `alt`

所以畫面不顯示名稱，但仍保留輔助資訊與偵錯可讀性。

### 4. 圖片不再被裁切

原本：

```css
object-fit: cover;
```

現在：

```css
object-fit: contain;
```

圖片會完整顯示，不會被卡牌邊框吃掉。

### 5. 移除 BOSS 區說明文字

原本 BOSS 區會顯示規格說明文字：

「整個背景是 BOSS 圖卡。BOSS 反擊前會出現 WARNING...」

現在已改為正式版顯示，不再出現測試文字。

### 6. 戰鬥中隱藏關卡按鈕列

戰鬥開始後會自動加上：

```css
.link-battle-shell.in-battle
```

並隱藏上方一整排關卡按鈕，讓戰鬥畫面更像正式遊戲介面。

## 檢查結果

已執行：

```bash
node --check js/link-battle.js
```

結果：通過。

也檢查 `index.html` 內嵌 script 語法，結果通過。

## 注意事項

本次主要修正前端正式版 UI，後端沒有變更。

連線規則仍維持原本設計：

- 同卡
- 同層
- 未被上層覆蓋
- 可直線連線
- 可一次轉彎
- 不可跨層
- 不可兩次以上轉彎

如果之後要做更接近上海麻將的「自由邊可消除」規則，那會是另一版規則調整。
